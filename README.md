# Academic Website (GitHub Pages)

This repository contains a Jekyll website using the **Minimal Mistakes** theme via `remote_theme`.

## Quick deploy

1. Create a GitHub repository named: `<your-username>.github.io`
2. Upload the contents of this folder to the repository (root).
3. In GitHub: **Settings â†’ Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / root
4. Wait for GitHub Pages to build and publish.

## Customize

- Edit `_config.yml` (title, description, links, avatar).
- Edit pages under `_pages/`.
- Replace the research map image at `assets/images/research-map.png`.
- (Optional) Add your photo as `assets/images/avatar.png`.
